## development

### Flask api

```sh
$ cd api
$ pip3 install --upgrade -r requirements.txt
$ python3 app.py
...
INFO:werkzeug: * Running on http://0.0.0.0:5000/ (Press CTRL+C to quit)
```

#### :whale: Run docker
```sh
$ docker-compose build
$ docker-compose up
Attaching to flask_api
```

access `localhost:5000`
